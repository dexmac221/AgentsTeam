# Tetris Game (Pygame)

## Description
A classic Tetris game implemented with Python and Pygame.  
Features:
- All 7 tetromino pieces (I, O, T, S, Z, J, L)
- Rotation, movement, and hard drop
- Line clearing with scoring
- Level progression and fall speed increase
- Pause functionality
- Game over handling

## Setup

### Prerequisites
- Python 3.8+
- pip
